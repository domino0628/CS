# API로 렌더링 vs 템플릿 엔진으로 렌더링의 차이? : api를 사용하면 서버는 데이터를 제공하고 클라이언트가 렌더링함
<br><br>
# 템플릿 엔진 렌더링은 return으로 html 반환하는 단순한 형태
<br><br>
# API에서 파라미터는 엔티티를 받는게 아니라 DTO형태로 받는게 좋다, 왜냐하면 API 스펙을 직관적으로 알 수 있기 때문이다.
