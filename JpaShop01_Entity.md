# ORM : ORM 기술은 객체(Object)와 DB테이블을 매핑하여 데이터를 객체화하는 기술
<br><br>
SQL문이 아닌 메소드 형식으로 직접 DB데이터 관리가 가능함
<br><br>

# Java Persistence Api : ORM의 일종으로, 프로그램 종료 후에도 영구 저장이 가능하다 
<br><br>
Hibernate는 구현체중 하나
<br><br>
